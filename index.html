Instalasi Source Code

  1. Klik tombol Download .zip besar di atas.
  2. Extract file zip tersebut.
  3. Buka Git Bash dari dalam folder hasil extract tadi.
  4. Jalankan perintah composer install, tunggu sampai selesai.
  5. Jalankan perintah mv .env.example .env, pastikan sekarang terdapat file .env di root aplikasi.
  6. Jalankan perintah php artisan key:generate, pastikan sekarang di dalam file .env bagian APP_KEY. terisi angka/huruf random.
  7. Di dalam file .env, ubah isian bagian DB_DATABASE, DB_USERNAME, dan DB_PASSWORD sesuai dengan keadaan sistem Anda.
  8. Jalankan perintah php artisan config:cache, pastikan berjalan sukses.
  9. Terakhir, jalankan perintah php artisan serve untuk menjalankan web server bawaan Laravel
  10. Buka web browser favorit Anda, arahkan ke alamat: http://localhost:8000

Tentang UTS

Secara singkat, UTS kita nanti adalah membuat Create, Read, Update, Delete untuk entitas:

  1. Customers
  2. Employees
  3. Shippers
  4. Suppliers

Untuk entitas Orders dan Order Details kerjakan HANYA Read (index dan show) saja.
